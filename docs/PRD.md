# Mamoji PRD（简版）

## 1. 产品定位

Mamoji 是面向个人与家庭的记账与预算管理系统，目标是帮助用户建立可持续的财务管理习惯。

## 2. 目标用户

- 个人用户: 管理个人收入、支出、预算
- 家庭用户: 多成员共享账本，查看整体收支结构

## 3. 核心问题

1. 记账门槛高，难以长期坚持
2. 预算执行缺乏实时反馈
3. 交易数据多，但洞察与行动建议不足
4. 风险交易（超预算、高额、异常频次）缺少及时提醒

## 4. 产品目标

- 提供低摩擦记账流程
- 提供按月/按年预算可视化
- 提供交易分析、洞察和建议
- 提供 AI 助手辅助解释数据与给出行动建议

## 5. 功能范围

### 5.1 记账与账户

- 账户管理（新增、编辑、删除、汇总）
- 交易流水（筛选、分页、退款、批量查看）
- 分类管理（收入/支出分类）

### 5.2 预算与报表

- 预算管理（周期、额度、分类预算）
- 概览报表（本月、本年、趋势、分类占比）
- 高级报表（年度视图、资产负债、对比分析）

### 5.3 账本与协作

- 账本管理
- 成员邀请与角色管理
- 默认账本能力

### 5.4 AI 助手

- 财务助手与股票助手两类场景
- 支持 Auto / Agent / LLM 模式
- 支持意图分类（预算/分类/流水/收支）提升工具选择精度

## 6. 非功能目标

- 响应性能: 常见查询 < 1s
- 可用性: 关键页面可在移动端可读可操作
- 可维护性: 核心模块有注释和文档支撑
- 安全性: JWT 鉴权、权限控制、风控规则拦截

## 7. 关键指标（建议）

- 次日留存
- 周记账次数
- 预算设置覆盖率
- 预算超支告警触发率
- AI 问答满意度（可通过反馈面板统计）

## 8. 近期迭代方向

1. 风控规则扩展（频次、金额、分类异常）
2. 报表可视化升级（年度对比、现金流预测）
3. AI 回答质量提升（结构化输出 + 工具校验）
4. 文档与注释持续补齐，降低维护成本
