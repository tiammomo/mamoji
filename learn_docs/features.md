# 功能详解

## 账户管理

### 添加账户
1. 点击右上角「添加账户」
2. 选择账户类型
3. 填写信息并保存

### 账户类型
| 类型 | 说明 | 计入净资产 |
|------|------|------------|
| 银行卡 | 储蓄卡、借记卡 | 是 |
| 信用卡 | 信用卡、贷记卡 | 否（显示为负数） |
| 数字钱包 | 支付宝、微信等 | 是 |
| 现金 | 现金余额 | 是 |
| 投资 | 股票、基金 | 是 |
| 负债 | 贷款、借款 | 否 |

### 账户操作
- **编辑** - 修改账户名称和余额
- **删除** - 删除账户（需先转移余额）

## 交易记录

### 添加交易
1. 点击「添加交易」按钮
2. 选择交易类型（收入/支出）
3. 选择分类（餐饮、交通等）
4. 输入金额、日期、备注
5. 点击保存

### 交易类型
- **收入** - 增加余额（工资、奖金等）
- **支出** - 减少余额（消费、转账等）

### 退款功能
支出记录支持退款：
1. 找到要退款交易记录
2. 点击「退款」按钮
3. 输入退款金额（不能超过原交易金额）
4. 填写退款原因（可选）
5. 确认退款

**注意**：退款金额会从已花费预算中扣除。

### 筛选和搜索
- 按日期范围筛选
- 按分类筛选
- 按账户筛选
- 关键词搜索备注

## 预算管理

### 创建预算
1. 进入「预算」页面
2. 点击「创建预算」
3. 填写预算信息：
   - 预算名称（如「餐饮预算」）
   - 预算金额（如 ¥2,000）
   - 开始日期
   - 结束日期
4. 点击保存

### 查看预算进度
- **进行中** - 预算在有效期内
- **超支** - 已花费超过预算
- **已取消** - 已停用的预算

### 预算状态
| 状态 | 说明 |
|------|------|
| 正常 | 使用率 < 80% |
| 警告 | 使用率 80%-100% |
| 超支 | 使用率 > 100% |

### 预算进度条
```
████████████░░░░  60%
¥1,200 / ¥2,000   剩余 ¥800
```

## 报表统计

### 按月查看
1. 默认显示上一个月数据
2. 点击月份下拉框选择其他月份
3. 可切换查看「自定义日期范围」

### 报表内容
- **收支概览** - 本月收入、支出、净收入
- **收入分布** - 各类收入占比饼图
- **支出分布** - 各类支出占比饼图
- **趋势图** - 每日收支折线图
- **资产负债表** - 资产和负债汇总

### 环比对比
按月查看时，会显示上月数据对比：
- 上月收入/支出/净收入
- 环比变化百分比

## 数据安全

### 备份建议
- 定期导出重要数据
- 重要操作前做好记录

### 数据存储
- 用户数据存储在 MySQL 数据库
- 账户余额实时更新
- 历史记录永久保存（除非删除）
