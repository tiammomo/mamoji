# Mamoji 用户指南

欢迎使用 Mamoji 记账系统！本指南帮助您快速上手。

## 快速开始

### 1. 注册账号
1. 打开应用 http://localhost:43000
2. 点击「注册账号」
3. 输入邮箱和密码完成注册

### 2. 添加账户
1. 进入「账户」页面
2. 点击「添加账户」
3. 选择账户类型（银行卡/信用卡/现金/数字钱包）
4. 输入账户名称和初始余额

### 3. 记录收支
1. 进入「交易」页面
2. 点击「添加交易」
3. 选择类型（收入/支出）
4. 输入金额、分类、日期和备注

### 4. 设置预算
1. 进入「预算」页面
2. 点击「创建预算」
3. 输入预算名称和金额
4. 设置起止日期

### 5. 查看报表
1. 进入「报表」页面
2. 选择要查看的月份
3. 查看收支分析和趋势图

## 功能说明

### 账户管理
- **银行卡** - 储蓄卡、借记卡
- **信用卡** - 信用卡、贷记卡
- **数字钱包** - 支付宝、微信、PayPal
- **现金** - 现金余额

### 分类管理
系统预置常用分类：
- **收入**: 工资、奖金、投资、其他收入
- **支出**: 餐饮、交通、购物、娱乐、生活费用

### 预算功能
- 设置月度/季度/年度预算
- 超支时会显示警告
- 支持查看预算使用进度

### 退款功能
支出记录支持部分退款：
- 点击交易记录的「退款」按钮
- 填写退款金额和原因
- 退款金额会返还到可用预算

## 常见问题

### Q: 忘记密码怎么办？
A: 目前暂不支持自助找回密码，请联系管理员。

### Q: 如何删除账户？
A: 账户删除后不可恢复，请确认后再操作。

### Q: 数据会丢失吗？
A: 建议定期导出重要数据。数据存储在 MySQL 数据库中。
