# Mamoji 记账系统

个人/家庭记账系统，支持多账户管理、预算控制、收支分析。

## 技术栈

- 后端：Java + Spring Boot 3.x + MyBatis-Plus
- 数据库：MySQL 8.0
- 前端：待定
