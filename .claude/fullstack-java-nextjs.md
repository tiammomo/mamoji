# fullstack-java-nextjs

## 定义
Java + Next.js 全栈开发专家 skill，处理前后端集成开发相关任务。

## 适用场景
- 前后端 API 对接
- 跨域配置处理
- 统一错误处理
- 认证流程 (JWT) 前后端配合
- 环境变量配置
- 前后端联调问题排查
- API 文档同步

## 执行步骤
1. 明确前后端各自职责
2. 确保 API 设计一致性（参考 docs/API.md）
3. 处理前后端数据类型映射
4. 协调认证/授权流程

## API 约定
- 基础路径: `/api/v1`
- 认证方式: JWT Token
- 响应格式统一:
  ```json
  {
    "code": 200,
    "message": "success",
    "data": {}
  }
  ```

## 环境配置
| 环境 | 后端端口 | 前端端口 |
|------|----------|----------|
| 开发 | 8080 | 3000 |
| 生产 | 按部署配置 | 按部署配置 |

- [ ] 请求/响应格式一致
