# Weather

Retrieve current weather conditions and forecasts for any location.

## Capabilities

- Current temperature and conditions
- Humidity levels
- Wind speed and direction
- UV index
- Air quality
- Multi-day forecasts
- Severe weather alerts

## Usage

Provide location information:
- City name (e.g., "Beijing", "New York")
- Coordinates (latitude, longitude)
- Airport code (e.g., "PEK", "JFK")

## Best Practices

- Specify units (Celsius/Fahrenheit)
- Check for weather-dependent activities
- Note relevant weather impacts
- Provide context for the user

## Notes

- Weather data sourced from external APIs
- Update times vary by provider
- Some features may require specific API keys
