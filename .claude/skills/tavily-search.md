# Tavily Search

Use Tavily to search the web for current information, news, and real-time data that may not be in my training data.

## When to Use

- When you need current information or latest news
- When you need real-time data or statistics
- When you need to look up recent events or developments
- When you need information beyond your training data cutoff

## Best Practices

- Formulate clear, specific search queries
- Use quotes for exact phrase matching
- Use site: to search specific websites
- Review multiple sources for accuracy
- Cite sources in your response

## Notes

- Tavily provides search results with source attribution
- Prefer official sources for accuracy
- Be mindful of information freshness
