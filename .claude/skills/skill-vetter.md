# Skill Vetter

Evaluate, discover, and recommend Claude Code skills for specific tasks.

## Capabilities

### Skill Discovery
- Find relevant skills for given tasks
- Search existing skills
- Identify skill gaps
- Recommend skill additions

### Skill Evaluation
- Assess skill quality
- Verify skill effectiveness
- Check skill documentation
- Evaluate skill fit for use case

### Skill Recommendations

**When to Recommend New Skills**
- Task requires specialized knowledge
- Existing skills insufficient
- Common patterns identified
- User requests specific capability

**How to Present Recommendations**
- Explain what the skill does
- Describe use cases
- Show how to install/use
- Provide alternatives if available

## Best Practices

1. **Know Available Skills** - Familiarize with current skill set
2. **Understand Use Cases** - Match skills to user needs
3. **Verify Quality** - Check skill documentation and relevance
4. **Suggest Incrementally** - Don't overwhelm with options
5. **Provide Context** - Explain why a skill is recommended

## Usage

When user asks:
- "How do I do X?"
- "Is there a skill for X?"
- "Can you find a skill for X?"
- Expresses interest in extending capabilities

Use the `find-skills` skill to search for relevant capabilities.
