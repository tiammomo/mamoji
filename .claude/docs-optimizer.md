# docs-optimizer

## 定义
这是一个专门用于优化 Markdown 文档的 skill。

## 执行步骤
1. 读取 docs 目录下的所有 .md 文件
2. 检查文档质量：
   - 链接完整性
   - 标题层级
   - 代码块格式
   - 表格格式
   - 错别字（中文常见）
3. 提供优化建议或直接修复

## 检查规则
- [ ] 链接是否有效（无破损链接）
- [ ] 标题是否从 h1 开始，层级是否递减
- [ ] 代码块是否有语言标记
- [ ] 表格是否对齐
- [ ] 是否有冗余空行
- [ ] 中文标点符号是否正确
- [ ] 英文和数字前后是否有空格

## 输出格式
返回优化报告，包含：
- 文件列表
- 发现的问题
- 优化建议
- 自动修复的内容（如果有）
